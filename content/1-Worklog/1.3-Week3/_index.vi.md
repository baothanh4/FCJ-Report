### Thành tựu Tuần 3:

* Hiểu cơ bản về bảo mật dữ liệu trên AWS, bao gồm:
  * Mã hóa và quản lý khóa với AWS KMS
  * Bảo vệ dữ liệu nhạy cảm với Amazon Macie
  * Quản lý chứng chỉ với AWS Certificate Manager
  * Chế độ bảo mật dữ liệu của S3, EBS, DynamoDB

* Học các khái niệm về bảo vệ lớp và ứng dụng:
  * Security groups, Elastic Load Balancers (ELB), AWS Regions
  * Cách hoạt động của AWS Shield và AWS WAF

* Hiểu vai trò của các dịch vụ trong việc phát hiện và phản hồi sự cố bảo mật:
  * Amazon Inspector, GuardDuty, Amazon Detective, AWS Security Hub

* Hiểu cơ bản về ngăn chặn truy cập trái phép:
  * IAM Identity Center (SSO)
  * AWS Secrets Manager
  * AWS Systems Manager
