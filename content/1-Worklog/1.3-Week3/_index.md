### Week 3 Achievements

#### 1. Data Protection and Governance
- Understood:
  - Encryption & key management using **AWS KMS**
  - Sensitive data discovery using **Amazon Macie**
  - Certificate lifecycle management with **AWS Certificate Manager**
  - Built-in security features of **S3**, **EBS**, and **DynamoDB**

#### 2. Network and Application Security
- Learned layered security concepts:
  - Security Groups
  - Elastic Load Balancers (ELB)
  - Multi-Region protection strategies
- Understood how **AWS Shield** and **AWS WAF** mitigate DDoS and web attacks.

#### 3. Threat Detection & Incident Response
- Understood the roles of:
  - **Amazon Inspector** (vulnerability scanning)
  - **Amazon GuardDuty** (threat detection)
  - **Amazon Detective** (investigation & analysis)
  - **AWS Security Hub** (centralized security posture)

#### 4. Preventing Unauthorized Access
- Learned fundamentals of:
  - **IAM Identity Center (SSO)**
  - **AWS Secrets Manager**
  - **AWS Systems Manager**
